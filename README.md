## About
- A lightweight mod to permanently lock baby animals' age, keeping them small and adorable forever. It has no external dependencies.

## Features
- Feed baby animals a Poisonous Potato to permanently prevent them from growing up (supports tadpoles).
- Name baby animals to lock their growth (configurable option).
- Use a Milk Bucket to remove the lock and allow normal growth.
- Configurable entity blacklist to exclude specific animals.
- Built-in configuration screen for easy customization.

## Config
- namingLock - Naming a baby animal will prevent it from growing up (default: false).
- entityBlacklist - Entities in this list will not be affected by the mod.
